# comobot Logo 提案

## 文件说明

- `comobot-mark.svg`：图标版（适合 favicon、App 图标、头像）
- `comobot-logo-horizontal-light.svg`：横版浅色底主标
- `comobot-logo-horizontal-dark.svg`：横版深色底主标

## 设计思路

- 图形以字母 `C` 的轨道感构成，表达 `comobot` 的智能循环与持续进化。
- 中心为简化机器人面部，强化产品属性（AI 助手）。
- 保持几何化与高辨识，便于在小尺寸下使用。

## 品牌建议

- 主色：`#2E6BFF`（智能蓝）
- 辅色：`#00C2FF`（活力青）
- 深色文字：`#111827`
- 次级灰：`#6B7280`

## 推荐使用

- 官网 Header：优先 `comobot-logo-horizontal-light.svg`
- 深色 Banner：使用 `comobot-logo-horizontal-dark.svg`
- 社媒头像 / 应用图标：使用 `comobot-mark.svg`

如果你想，我下一步可以再做两版风格分支：
1. 更极简（仅字母 C + 一点机器人特征）
2. 更未来感（增加轻微霓虹与科技线条）
